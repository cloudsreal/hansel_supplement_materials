# HANSEL Supplement Materials
 
This repository contains supplementary materials for the user study conducted in the HANSEL project.

## Contents
 
- **prompt.md**: LLM prompts used in the HANSEL pipeline.
- **study/User Study Protocol.pdf**: The complete protocol for the user study.
- **study/analysis/**: Raw and derived datasets and the Jupyter notebook used for quantitative analysis.
- **study/survey/**: Recruitment, pre-study, task response, and post-study surveys.
